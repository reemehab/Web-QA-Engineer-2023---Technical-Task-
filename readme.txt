This project tests two distinct components:

### 1. My Store Website Testing with NightwatchJS
http://automationpractice.multiformis.com/index.php

### 2. API Testing with mock-user-auth and supertest and jest.
"https://www.npmjs.com/package/mock-user-auth"
Situated in the `APIS` directory, this section is dedicated to testing APIs Further insights into this component are available in its dedicated README file within the `APIS` directory.

For more detailed instructions and information about each project, please refer to the respective README files in their corresponding directories.
